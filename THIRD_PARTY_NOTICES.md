## Salamander Grand Piano V3 Samples

This plugin bundles a compact MP3 subset of Salamander Grand Piano V3 samples for offline piano playback.

The bundled files are MP3 samples hosted by the Tone.js project and are embedded into the built plugin bundle as data URLs. This plugin uses a reduced subset of the source sample set rather than the full Salamander Grand Piano library.
